Phase 1 — Cockpit safety check

Initial safety and power‑off condition
Safety
▶

1.1 Aircraft condition

Parking / chocks
  • Parking brake — SET
    Ensure aircraft is held before power‑up.
  • Wheel chocks — IN PLACE (if available)
    Extra security at the stand.

1.2 Controls & switches

Safe positions
  • Thrust levers — IDLE / SHUTOFF
    No thrust on power‑up.
  • Speedbrake lever — DOWN
    Spoilers retracted.
  • Flap lever — UP
    Clean configuration.

1.3 Gear & panel

Basic checks
  • Landing gear lever — DOWN
    Confirm down‑locked indication.
  • Battery switches — OFF (if arriving to cold & dark)
    Confirms true cold & dark state.

Phase 2 — Power‑up & preliminary preflight

Electrical power, basic system bring‑up
Power‑up
▶

2.1 Electrical power

ELEC / APU
  • Battery switches 1 & 2 — ON
    Energizes essential DC buses.
  • External power — CONNECT (if available)
    Preferred for stable initialization.
  • If no EXT PWR: APU — START
    APU MASTER → ON · START → PRESS · Wait APU AVAIL · APU GEN → ON.
Verify: ELEC synoptic shows all buses powered · No ELEC ESS / DC BUS cautions

2.2 IRS / navigation alignment

IRS / FMS
  • IRS mode selectors L/C/R — NAV
    Begin alignment on all three units.
  • FMS POS INIT — Set GPS / gate position
    Improves navigation accuracy.
Verify: IRS ALIGN countdown active · No IRS FAULT messages

2.3 Fire & warning tests

Safety
  • Fire detection test — PERFORM
    FIRE TEST switch → HOLD · Confirm aural, lights, master warning.
  • Master warning / caution — TEST / RESET
    Confirms alerting system operation.
Verify: All test indications extinguish after release

Phase 3 — Cockpit preparation

Panels, fuel, hydraulics, air, FMS
Prep
▶

3.1 Fuel panel

FUEL
  • Fuel quantity — CHECK
    Matches planned fuel load.
  • Fuel pumps — ON (tanks with fuel)
    Ensures pressurized feed to engines.
  • Crossfeed — AUTO
    Normal MD‑11 configuration.
Verify: FUEL synoptic pumps green · No FUEL PRESS cautions

3.2 Hydraulic systems

HYD
  • Hydraulic systems 1 / 2 / 3 — ON
    Pressurize primary flight controls and gear.
  • Aux hydraulic pump — ON (if needed)
    Builds brake pressure on ground if required.
Verify: HYD synoptic pressurized · Brake accumulator in green band

3.3 Air conditioning & pneumatics

BLEED / PACKS
  • Packs — AUTO
    Normal cabin conditioning.
  • Recirc fans — ON
    Improves airflow and efficiency.
  • Temperature selectors — SET
    Cabin and cockpit comfort.
  • APU bleed — ON (if using APU)
    Provides bleed air for packs and engine start.
Verify: CABIN synoptic stable · No PACK / BLEED cautions

3.4 FMS & performance

FMS / PERF
  • FMS IDENT — CHECK
    AIRAC validity and aircraft type.
  • POS INIT — SET
    Use GPS or gate coordinates.
  • ROUTE — ENTER
    Origin, destination, route, SID/STAR as required.
  • PERF INIT — ENTER
    ZFW, CG, cruise level, cost index, reserves.
Verify: No FMS INCOMPLETE message · VNAV/ATS available

3.5 MCP & radios

MCP / COM / NAV
  • COM1/2 — SET
    ATIS, ground, clearance frequencies.
  • NAV1/2 — SET
    Departure navaids or ILS as needed.
  • MCP heading — RUNWAY / SID
    Runway heading or initial track.
  • MCP altitude — INITIAL CLEARANCE
    Per ATC or SID restriction.
  • MCP speed — V2 (later)
    Set after takeoff performance computed.
Verify: FD ON both sides · ATS ARMED (white)

Phase 4 — Before start

Doors, pushback, final checks
Before start
▶

4.1 Cabin & doors

Cabin secure
  • Doors — CLOSED & ARMED
    All passenger and cargo doors closed.
  • Seat belts / no smoking — ON
    Cabin ready for pushback.

4.2 Takeoff data

Perf / trim
  • V1 / VR / V2 — SET
    From performance calculation / FMS.
  • MCP speed — V2
    Set for takeoff.
  • Stab trim — SET
    Per FMS / performance sheet.

4.3 Final checks

Before engine start
  • Beacon — ON
    Signals engine start imminent.
  • Parking brake — SET
    Hold aircraft for start / pushback.
  • Pushback / start clearance — OBTAINED
    From ATC or ground crew.
Verify: EICAS shows no critical cautions · Ground crew ready

Phase 5 — Engine start

FADEC‑controlled start, 2 → 1 → 3
Start
▶

5.1 Preconditions

Before first start
  • Hydraulic systems 1 / 2 / 3 — ON
    Flight controls and brakes powered.
  • Fuel pumps — ON
    All tanks with fuel pressurized.
  • APU bleed — ON (if using APU)
    Provides pneumatic pressure for start.
  • Packs — OFF
    Maximizes pneumatic pressure.
  • Thrust levers — IDLE
    Prevents unintended thrust.
  • ATS — ARMED
    White indication on FMA.
Verify: No HYD / FUEL / BLEED warnings · Pneumatic pressure ≥ ~30–40 PSI

5.2 Engine 2 start

Normal first engine
Normal order: 2 → 1 → 3
FADEC controls fuel
  • Engine start selector — ENG 2
    Select center engine.
  • Engine 2 start switch — START
    N2 should begin to rotate.
Monitor: N2 rising · At ~15% N2 FADEC introduces fuel · EGT rises · N1 spools

5.3 Monitor engine 2

EICAS
  • N2 — 20–50%
    Smooth acceleration, no stagnation.
  • EGT — Within limits
    Abort if hot start (rapid EGT rise).
  • Oil pressure — RISING
    Abort if no oil pressure.
Abort if: Hot start · Hung start · No oil pressure

5.4 Engine 2 stabilized

Idle
  • Confirm idle parameters
    N1 ~20–25% · N2 ~60% · EGT stable · Oil pressure green.
  • Engine start selector — OFF
    End engine 2 start sequence.

5.5 Engines 1 and 3

Repeat flow
  • Engine 1 — START
    Selector → ENG 1 · START → Monitor · Selector → OFF when stabilized.
  • Engine 3 — START
    Selector → ENG 3 · START → Monitor · Selector → OFF when stabilized.
Verify: All three engines stable at idle · No engine / bleed cautions

5.6 After start

Post‑start config
  • Packs — AUTO
    Restore cabin conditioning.
  • APU bleed — OFF
    Engines now supply bleed air.
  • APU — OFF (after cooldown)
    Shut down when no longer needed.
  • Aux hydraulic pump — OFF (if not required)
    Engine‑driven pumps now supply pressure.
  • Flight controls — CHECK
    Full, free, correct movement.
  • Slats / flaps — SET FOR T/O
    Per performance / FMS.
Verify: No BLEED / PACK / HYD cautions · Takeoff config OK

Phase 6 — Taxi

From stand to runway
Taxi
▶

6.1 Taxi out

Initial movement
  • Taxi clearance — OBTAINED
    From ATC or ground.
  • Taxi lights — ON
    Indicates movement to others.
  • Parking brake — RELEASE
    Begin taxi.

6.2 During taxi

Checks
  • Brakes — CHECK
    Verify normal braking response.
  • Nosewheel steering — CHECK
    Smooth directional control.
  • Flight instruments — CHECK
    Heading and track follow taxi path.

Phase 7 — Before takeoff

At or near runway
Before T/O
▶

7.1 Final configuration

Takeoff config
  • Flaps / slats — CHECK SET
    Matches planned takeoff setting.
  • Trim — CHECK
    Within green band, per performance.
  • Spoilers — ARMED
    For automatic deployment on landing / rejected takeoff.

7.2 Lights & modes

Runway entry
  • Landing lights — ON
    When entering runway.
  • Strobe lights — ON
    Indicates runway occupancy.
  • Transponder — TA/RA
    Set to ALT / TA‑RA for departure.
Verify: Takeoff config warning — NO WARNING

Phase 8 — Takeoff

Roll, rotation, initial climb
Takeoff
▶

8.1 Takeoff roll

Runway
  • Thrust — SET T/O
    Use ATS or manual to planned thrust.
  • 80 kts — CALLED
    Cross‑check airspeed indications.
  • V1 — CALLED
    Decision speed.
  • VR — ROTATE
    Rotate to target pitch.

8.2 Initial climb

After liftoff
  • Positive rate — GEAR UP
    On “positive climb” call.
  • Climb thrust — SET
    At acceleration altitude, per FMS / procedure.
  • Flaps — RETRACT ON SCHEDULE
    Follow FMS / speed schedule.

Phase 9 — Climb & cruise

Climb checks, cruise monitoring
Climb / cruise
▶

9.1 Climb

After clean‑up
  • Landing lights — OFF (above transition)
    Per company / local procedures.
  • Seat belt sign — AS REQUIRED
    When safe and stable.

9.2 Cruise

Level at cruise
  • Cruise level — CONFIRM
    Altimeters set to standard / local as appropriate.
  • Fuel / systems — MONITOR
    Cross‑check fuel burn, engine parameters, pressurization.

Phase 10 — Descent & approach

Descent planning, approach setup
Descent / approach
▶

10.1 Descent planning

Top of descent
  • Arrival / approach — SELECT
    STAR and approach in FMS.
  • Landing data — SET
    Landing weight, Vref, autobrake.
  • Altimeters — SET
    Transition level / local QNH.

10.2 Approach setup

Before final
  • NAV radios — SET
    ILS / navaids as required.
  • MCP — SET
    Heading, altitude, approach mode as needed.
  • Seat belt sign — ON
    Prepare cabin for landing.

Phase 11 — Landing

Final approach, touchdown, rollout
Landing
▶

11.1 Final approach

Configured to land
  • Flaps — LANDING SETTING
    Per landing performance / FMS.
  • Gear — DOWN
    Three green indications.
  • Landing lights — ON
    On final approach.

11.2 Touchdown & rollout

After touchdown
  • Spoilers — DEPLOY (auto)
    Confirm spoiler deployment.
  • Reverse thrust — AS REQUIRED
    Within limits, cancel by ~60 kts.
  • Braking — AS REQUIRED
    Manual or autobrake as planned.

Phase 12 — After landing

Runway vacated, taxi to stand
After landing
▶

12.1 Cleanup

Runway exit
  • Spoilers — DISARM
    Return speedbrake lever to DOWN.
  • Flaps — UP (or as required)
    Retract after clear of runway.
  • Landing lights — OFF
    Taxi lights remain ON.

12.2 Taxi in

To gate / stand
  • Transponder — STBY (per ATC)
    After vacating runway.
  • APU — START (if required)
    For electrical / air at gate.

Phase 13 — Shutdown & secure

Engines off, aircraft secured
Shutdown
▶

13.1 Shutdown

At stand
  • Parking brake — SET
    Before engine shutdown.
  • External power — CONNECT (if available)
    Stable power source at gate.
  • Thrust levers — SHUTOFF
    Shut down all engines.

13.2 Post‑shutdown

Systems
  • Seat belt sign — OFF
    Passengers may disembark.
  • Fuel pumps — OFF
    When engines and APU not requiring fuel.
  • Hydraulic pumps — OFF
    Depressurize systems as appropriate.

13.3 Secure aircraft

Cold & dark
  • APU — OFF (if not needed)
    After cool‑down and external power available.
  • External power — AS REQUIRED
    Leave connected or disconnect per turnaround / shutdown.
  • Battery switches — OFF (for full cold & dark)
    Only if aircraft is to be fully powered down.
Verify: All displays dark (if cold & dark) · Aircraft secured
